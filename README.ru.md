> Профессиональный интерактивный скрипт настройки сервера для Debian/Ubuntu VPS

**Языки:** [English](README.md) | [Русский](README.ru.md)

Однофайловый Bash-скрипт с интерактивным меню для первоначальной настройки VPS. Без внешних зависимостей, без Python, без диалоговых окон — только чистый Bash.

## Возможности

- **16 независимых пунктов меню** — запускайте только то, что нужно
- **Профессиональная обработка ошибок** — скрипт никогда не падает, всегда возвращает в меню
- **ANSI-цветной вывод** — современный терминальный интерфейс
- **Готов к curl-bash** — установка в одну команду
- **Идемпотентность** — безопасно запускать несколько раз

## Поддерживаемые ОС

- Debian 12+
- Ubuntu 22.04+
- Ubuntu 24.04+

## Быстрый старт

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/amirim1/vps-quickstart/main/setup.sh)
```

Или скачайте и запустите локально:

```bash
curl -fsSL -o setup.sh https://raw.githubusercontent.com/amirim1/vps-quickstart/main/setup.sh
bash setup.sh
```

## Меню

| # | Функция | Описание |
|---|---------|----------|
| 1 | Обновление системы | apt update, upgrade, autoremove, autoclean |
| 2 | Установка базовых пакетов | curl, wget, git, jq, htop, btop, vim и др. |
| 3 | Настройка SSH | Смена порта, отключение пароля/root, ключи |
| 4 | Настройка Firewall (UFW) | Установка, порты, включение с подтверждением |
| 5 | Установка Fail2Ban | Авто-синхронизация с текущим портом SSH |
| 6 | Создание Swap | Размер от пользователя, проверка диска, fstab |
| 7 | Включение BBR | Проверка ядра, настройка sysctl |
| 8 | Управление IPv6 | Отключение/включение/проверка с синхронизацией UFW |
| 9 | Информация о сервере | ОС, CPU, RAM, диск, сеть, виртуализация |
| 10 | Проверка сети | Ping, DNS, HTTP/HTTPS |
| 11 | Проверка скорости | Скачивание/загрузка через speedtest-cli |
| 12 | Проверка домена | A/AAAA-записи, совпадение IP, HTTPS |
| 13 | Установка 3x-ui | Панель MHSanaei 3x-ui |
| 14 | Создание пользователя | Имя, пароль, sudo, SSH-ключ |
| 15 | Настройка Sudo | Sudo без пароля с проверкой visudo |
| 16 | Выход | Чистый выход |

## Архитектура

```
setup.sh
├── Configuration Section    # Все параметры в одном месте
├── Utility Functions        # Логирование, проверки, хелперы
├── Validation Functions     # Root, ОС, архитектура, ядро
├── Feature Functions (1-15) # Независимые, самодостаточные
└── Main Loop              # Отображение меню и выбор
```

## Логирование

Все операции записываются в `/var/log/vps-quickstart.log` в виде обычного текста (без ANSI-кодов).

## Требования

- Root-доступ
- Bash 5.0+
- Интернет-соединение (для некоторых функций)
